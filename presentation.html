Core Contribution

VAD-Guided Dynamic Wiener Filter

STEP 1

Dual-Feature VAD

Speech detected if Log-Energy is HIGH and Spectral Entropy is LOW.

STEP 2

Dynamic Noise PSD Update

Updates noise profile only during silence frames using exponential average.

STEP 3

Adaptive Oversubtraction β(t)

Dynamically scales β between 1.0 (clean) and 3.0 (noisy) based on frame SNR.

// Adaptive Gain Function
H(f,t) = max(|Y(f,t)|² - β(t)·P_nn(f,t), 0) / |Y(f,t)|²

★ The Innovation: Continuous noise tracking without speech contamination, eliminating musical noise artifacts.
